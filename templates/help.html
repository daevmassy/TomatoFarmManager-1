üå± Planting

Managing Your Plants

Add New Planting:

  1. Click the "Add New Planting" section
  2. Enter variety (e.g., Cherry, Roma, Beefsteak)
  3. Set planting date and expected harvest date
  4. Specify field location and quantity
  5. Select status (Growing, Harvested, Removed)
  6. Add optional notes
  7. Click "Add Planting" button

Tips:

  • Use clear field location names for easy tracking
  • Expected harvest date helps with planning
  • Update status regularly to keep records accurate

üß∫ Harvesting

Recording Harvest Data

Add Harvest Record:

  1. Select the plant (optional - can be general harvest)
  2. Enter harvest date
  3. Input quantity and select unit (kg/tons)
  4. Choose quality grade (Grade A, B, or C)
  5. Add notes about harvest conditions
  6. Submit the record

Quality Grades:

  • Grade A: Premium quality, best for retail
  • Grade B: Good quality, standard retail
  • Grade C: Processing grade, for sauces/products

üì¶ Inventory

Managing Supplies & Equipment

Add Inventory Item:

  1. Enter item name (e.g., "NPK Fertilizer")
  2. Select category (Seeds, Fertilizer, Equipment, etc.)
  3. Input current quantity and unit
  4. Set minimum quantity threshold
  5. Add supplier information
  6. Save the item

Low Stock Alerts:

  • Items below minimum quantity are highlighted in red
  • Check inventory regularly to avoid shortages
  • Update quantities after use or restocking

Tracking Farm Activities

Record Operations:

  2. Enter operation date
  3. Specify field location

üí∞ Sales

Managing Sales Transactions

Record a Sale:

  1. Enter sale date
  2. Input customer name (optional)
  3. Enter quantity sold and unit
  4. Set price per unit
  5. Total amount calculates automatically
  6. Select payment status (Paid/Pending)

Tips:

üë∑ Employee Tasks

Assigning & Tracking Work

Assign a Task:

  1. Select employee number (1-10)
  2. Choose task date
  3. Select task type
  4. Specify field location
  5. Add detailed description
  6. Set start time and estimated hours

Task Management:

  • Pending: Task assigned but not started

  • Completed: Task finished
  • Employees can start/finish their own tasks
  • Admins can update task status or delete tasks

‚ùì Common Questions

Frequently Asked Questions

Q: What's the difference between Admin and Guest login?

A: Admins have full access to add, edit, and delete records. Guests (employees) can only view information but cannot make changes.

Q: How do I change the theme?

A: Use the toggle switch in the header (next to the logout button) to switch between light and dark themes. Your preference is saved automatically.

Q: Can I delete a record?

A: Yes, if you're logged in as Admin. Each record has a Delete button. Note: Deletion is permanent!

Q: Why are some inventory items highlighted in red?

A: Red highlighting indicates low stock - the quantity is at or below the minimum threshold you set.

Q: How do I link a harvest to a specific plant?

A: When adding a harvest record, select the plant from the dropdown. You can also leave it unselected for general harvests.

Q: Can I export or print reports?

A: Currently, you can use your browser's print function (Ctrl+P or Cmd+P) to print or save as PDF.
